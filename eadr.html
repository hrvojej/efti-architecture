eADR Sustav — Detaljan opis

NPOO C1.4. R1-I5 — Integralni nacionalni sustav za praćenje prijevoza opasnih tvari (DG) u cestovnom prometu.

🛡️ Tri funkcionalna područja

eADR Sustav (AKD d.o.o.)

🎓

(A) Upravljanje osposobljavanjem

Digitalizirani sustav za cjelokupni životni ciklus ADR osposobljavanja — od prijave na tečaj do izdavanja i obnove ADR potvrde.

7 programa osposobljavanja:

  • Osnovno ADR (sve klase osim 1 i 7)
  • Specijalizacija — klasa 1 (eksplozivne tvari)
  • Specijalizacija — klasa 7 (radioaktivni materijali)
  • Cisterne (dodatno osposobljavanje)
  • Obnova osposobljavanja (svaka 5 godina)
  • Sigurnosni savjetnik — osnovno
  • Sigurnosni savjetnik — obnova
📜

(B) Elektronička dopuštenja

Digitalno izdavanje, praćenje i upravljanje ADR dopuštenjima za prijevoz specifičnih klasa opasnih tvari.

4 vrste ADR dopuštenja:

  • Klasa 1 — Eksplozivi (MUP)
  • Klasa 7 — Radioaktivni mat. (MUP-RCZ)
  • Posebne rute — Ograničene rute (MMPI)
  • Izvanredni — Posebni uvjeti (MMPI/MUP)
🛣️

(C) eFTI Integracija (Nadzor)

Operativni nadzorni modul — primarni use case za inspektore tijekom cestovnih kontrola.

Ključne integracije:

  • eFTI Gate mreža (Uredba 2024/1942)
  • Affinis DWH — tahografski podaci (T+1 batch)
  • 8 nacionalnih registara
  • Cross-validacija eFTI vs. registri

📚 Baza podataka

8 Nacionalnih registara

🚛

1. ADR vozila

~3.000 aktivnih ADR vozila u HR. Registar certifikata, klasa, datuma isteka.

👤

2. ADR vozači

ADR potvrde osposobljenosti — klase, datumi, status, 5-godišnji ciklus.

🎓

3. Sigurnosni savjetnici

Registar certificiranih ADR sigurnosnih savjetnika.

🏫

4. Učilišta

Ovlaštena učilišta za ADR programe osposobljavanja.

📋

5. Ispitni centri

HAK i drugi ovlašteni centri za ADR ispite.

📜

6. Dopuštenja

Klasa 1 (eksplozivi), klasa 7 (radioaktivni), posebna dopuštenja.

🏢

7. Prijevoznici

Dijeljeno s Taho/NSCP — registar ADR prijevoznika i licenci.

📝

8. Nadzori

Pregledi, mjere, rezultati inspekcija — imutabilni zapisi.

📡 Komunikacijska arhitektura — sve integracije

# Veza Smjer Protokol Autentikacija Svrha
1 eADR ↔ NIAS Dvosmjerna SAML 2.0 / OAuth 2.0 X.509 Autentikacija službenika (MUP, DI, Carina, CVH)
2 eADR ↔ AFFINIS DWH Dvosmjerna REST API + CDC SSH Key / API Taho T+1, analitički export, KPI
3 eADR ↔ HR eFTI Gate Dvosmjerna eDelivery AS4 mTLS UIL upload + selective/full pull DG podataka
4 HR Gate ↔ EU Gateovi Gate-to-Gate eDelivery AS4 mTLS Prekogranična razmjena — strane platforme
5 eADR ← CVH Ingest REST/SFTP Certifikat ADR pregledi vozila, status odobrenja
6 eADR ← MMPI Ingest REST/SFTP Certifikat Registar prijevoznika, licenci, učilišta
7 eADR ← Učilišta/HAK Unos Web UI / API NIAS SSO Rezultati tečajeva i ispita

⚖️ Razgraničenje — eADR vs. NSCP

Aspekt eADR NSCP
Fokus Opasne tvari (DG) — ADR Sav cestovni teret (uklj. ne-DG)
Registri ADR vozila, vozači, potvrde, dopuštenja Prijevoznici, vozila, licencije (šire)
eFTI pretraga eFTI1451=true (samo DG) Sav teret
Nadzor ADR kontrole, oprema, klase Opći cestovni nadzor
Gate Dijele isti HR eFTI Gate
Taho podaci Da — Affinis DWH (T+1) Da — Affinis DWH (T+1)
SOTAH direktno NE NE

Ključno: eADR i NSCP dijele isti HR eFTI Gate — nisu dva zasebna Gatea. eADR je specijalizirani modul koji se fokusira na DG pošiljke (eFTI1451=true) unutar istog eFTI okruženja.
